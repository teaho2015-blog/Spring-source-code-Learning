****##2.2.1事务
事务就是以可控的方式对数据资源进行访问的一组操作。

**事务本身持有4个限定属性**(简称ACID属性)，来保证事务前后，系统始终处于“正确”的状态：

1. 原子性(Atomicity)

2. 一致性(Consistency)

3. 隔离性(Isolation)
    
    隔离级别：
    + Read Uncommitted
    
    + Read Committed
    
    + Repeatable Read
    
    + Serializable
    
    相关问题：
    
    + 脏读(dirty Read)
    
    + 不可重复读(Non-Repeatable Read)
    
    + 幻读(Phantom Read)
    
4. 持久性(Durability)
